# File Downloader SDK

文件下载SDK，支持断点续传、进度跟踪和状态管理。

## 快速开始

```bash
# 安装依赖
pnpm install

# 构建 SDK
pnpm build

# 运行示例
pnpm dev

# 构建示例
pnpm build:example
```

## 目录结构
```
.
├── src/              # SDK源码
├── examples/         # 示例项目
│   └── vue/         # Vue示例
├── __tests__/       # 测试文件
└── dist/            # 构建输出
```
